sliderdns-altering в качестве параметров принимает "hostname" и "ip".
Если ip нет, то оно берется из rem_addr.
Аутентификация производится http-сервером, скрипту нужно только username.

В конфиге для каждого юзера нужно обязательно указывать список hostname, разрешенных для изменения,
опционально - список разрешенных сетей для каждого hostname и признак оповещения на мыло, если для
этого юзера мыло задано.

Справка по ошибкам: http://docstore.mik.ua/orelly/linux/cgi/ch03_03.htm
